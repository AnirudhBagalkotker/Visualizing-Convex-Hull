Jarvis March Documentation Tutorial

Select the Points for which you want to find the convex hull or just press the random to generate random points.

Click on Simulate if you wish to simulate the Convex Hull creation. If you Simulation is selected then the Jarvis March algorithm starts to simulates each iteration of the algorithm. If you just want the result, press compute.

The Simulation now performs each iteration separately and is displayed on to the canvas.

The Convex Hull is entirely computed and is now displayed on to the canvas.
