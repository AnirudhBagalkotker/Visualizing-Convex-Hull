Jarvis March Documentation Tutorial

Select the Points for which you want to find the convex hull or just press the random to generate random points.

Click on Simulate if you wish to simulate the Convex Hull creation. If you Simulation is selected then the Kirkpatrick-Seidel algorithm starts to simulates and displays the Lower Hull of the points. If you just want the result, press compute.

The Simulation now computes the Upper Hull. The Bridge is also computed in this step which can be seen in the canvas by comparing the previous Lower Hull with the current Upper Hull.

The Convex Hull is entirely computed by joining the Upper and Lower Hull and is now displayed on to the canvas.
